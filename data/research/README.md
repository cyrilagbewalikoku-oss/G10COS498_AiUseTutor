# AI Agent Education Research (Curated)

This directory holds SAGE's pedagogical grounding — short, citable summaries of the research behind SAGE's teaching moves. When a learner asks *why* SAGE approaches something a particular way, skills can cite one of these entries directly.

Per spec §4c, this is the "External Data Source: AI Agent Education Research" RAG. In v2 CLI this is just a folder of markdown that skills can Read; on CollaborAITE it would be indexed into the RAG database.

## Files

- `craft-origins.md` — origin and empirical support for the CRAFT prompt framework SAGE teaches.
- `learner-predicts.md` — the "Learner Predicts" / cognitive-conflict pattern SAGE uses inside its NUDGE step.
- `ai-misconceptions.md` — catalog of common learner misconceptions about AI agents, and what each signals about where to intervene.
- `impact-audit.md` — the IMPACT framework for auditing AI use in an academic context.
- `merrill-bloom-summary.md` — one-page digest of Merrill's First Principles and Bloom's Revised Taxonomy as SAGE applies them.

## When skills should cite these

Use sparingly — this is research grounding, not a lecture hall. Skills should reach for these when:

- A learner asks "why does this work" or "is this actually backed by research"
- An Advanced or Critical Thinker learner wants deeper theoretical context
- A teacher or policy-designer learner wants to take a framework back to their own classroom

Do **not** read these at learners who haven't asked for them. The pedagogy is in the doing, not the citation.
